# OneHop 🚀

## Overview

OneHop is a travel assistance and trip-planning mobile application that helps users access travel information and AI-powered assistance through a single platform.

## Features

* User Authentication (Signup/Login)
* AI Travel Assistant
* Chat Functionality
* User Profile Management
* Cloud Hosted Backend

## Tech Stack

### Frontend

* React Native
* Expo
* TypeScript

### Backend

* FastAPI
* Python

### Database

* MongoDB

### Deployment

* Render
* UptimeRobot

## Setup Instructions

### Mobile App

1. Navigate to OneHopMobile
2. Install dependencies using npm install
3. Run using npx expo start

### Backend

1. Navigate to OneHop_Backend
2. Install dependencies
3. Start FastAPI server
